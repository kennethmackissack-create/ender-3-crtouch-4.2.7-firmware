# ender-3-crtouch-4.2.7-firmware
This is firmware for the Creality Ender 3 4.2.7 motherboard with CR/BL touch. This firmware was made using marlin and with the intention of making the Ender 3 more compatible with OctoPrint by enabling Host Action Commands. Please keep in mind that I am a complete beginner and have no experience with making firmware. I hope this firmware helps you.
